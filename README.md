# Finora App

Vue 3 + Vite frontend application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── assets/       # Static assets
├── components/   # Reusable Vue components
├── views/        # Page components
├── router/       # Vue Router configuration
├── stores/       # Pinia stores (if used)
├── App.vue
└── main.ts
```
